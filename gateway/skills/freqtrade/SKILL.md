# Freqtrade Skill

AI-powered trading strategy assistant for CryptoClaw.

## Features

- Natural language strategy creation
- Backtest execution and analysis
- Trading signal generation
- Portfolio management

## Usage

```
Run a backtest on BTC/USDT with RSI strategy
```

## Configuration

Set these environment variables:
- `FREQTRADE_API_URL`: Freqtrade REST API URL
- `FREQTRADE_API_KEY`: API key for authentication
