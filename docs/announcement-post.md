## 线程帖 1 - 商业模式

---

**商业模式：按结果付费**

传统量化工具：
- 月费 $99 起 ❌
- 不管赚没赚都要付 ❌

CryptoQClaw：
- 完全免费 ✅
- 有利润才付 10% ✅
- 高水位机制 ✅

**什么是高水位？**

简单说：只为"新利润"付费

举例：
- 1月赚 $500 → 付 $50
- 2月亏 $200 → 付 $0
- 3月赚 $100 → 付 $0 (还在回本)
- 4月赚 $300 → 付 $20 (只有 $200 是新利润)

**你的利益 = 我的利益**

你不赚钱，我不收费。

这才是 AI 时代该有的商业模式。

## 线程帖 2 - 隐私优先

---

**为什么数据必须本地存储？**

交易所 API Key 是你的命根子。

上传到服务器？
- 服务器被黑 → Key 泄露 → 资金归零
- 公司跑路 → Key 失控 → 后患无穷

CryptoQClaw 的做法：

🔐 **本地优先架构**
- API Key 只存在你的电脑
- AES-256 加密存储
- 永不上传云端

📱 **对话在 Telegram**
- 只是发指令
- 敏感数据不动

☁️ **云服务极简**
- 只存账单摘要
- 只确认支付
- 不碰你的 Key

**你的钱，你的数据，你掌控。**
